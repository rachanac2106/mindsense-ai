# 🧠 MindSense AI — Student Mental Wellness Portal

> An AI-powered emotional wellness dashboard for university students, featuring real-time facial emotion detection, an AI mental health chatbot, mood journaling, and guided breathing exercises.

---

## 🌐 Live Demo
👉 [rachanac2106.github.io/mindsense-ai](https://rachanac2106.github.io/mindsense-ai)

---

## ✨ Features

- **🎥 Live Emotion Detection** — Webcam-based facial recognition that detects Happy, Sad, Stressed, Anxious, and Neutral states in real time
- **🤖 AI Mental Wellness Chatbot** — Powered by Claude AI (Anthropic), provides empathetic, context-aware support
- **📊 Mood Analytics Dashboard** — 14-day emotion trend chart with interactive bar visualization
- **📝 Mood Journal** — Log personal entries with mood tags, stored locally
- **🫁 Guided Breathing Exercise** — Box breathing (4-4-4-4) with animated visual timer
- **🆘 SOS Crisis Support** — Instant access to iCall and Vandrevala Foundation helplines (India)
- **🌙 Dark / Light Mode** — Full theme toggle
- **📄 Wellness Report Export** — Download a personalized text report

## 🎯 Use Case

Built for university students dealing with academic pressure, exam stress, and emotional burnout. MindSense provides a private, judgment-free space to track and understand their mental wellness over time.
